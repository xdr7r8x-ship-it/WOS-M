# CAPTCHA Model Files

This directory should contain the ONNX captcha model from whiteout-project/bot.

## Files Required

1.  - The trained ONNX model
2.  - Model configuration (already created)

## How to Get the Real Model

The actual  file should be obtained from:
https://github.com/whiteout-project/bot/tree/main/models

Download  from the repository.

## Note

The model is a CNN-based captcha solver trained to recognize 4-character captchas.
It achieves ~97.9% accuracy on the validation set.
